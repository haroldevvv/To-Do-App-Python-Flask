About To-Do List Web App

Overview

This web application is a simple to-do list that allows users to keep track of their tasks. You can easily add, mark as complete, and delete tasks to stay organized.

Features

  • Task Creation: Add new tasks with a title and description.
  • Task Completion: Mark tasks as complete with a single click.
  • Task Deletion: Remove tasks that are no longer needed.

How to Use

1. Navigate to the homepage to view your current tasks.

2. Add a new task by filling out the form at the top of the page.

3. Click "Mark as Complete" to indicate a finished task.

4. Delete tasks you no longer need by clicking "Delete" next to them.

Technologies Used

  • Flask: The backend web framework used to build the application.
  • SQLite: A lightweight database for storing tasks.
  • Bootstrap: A front-end framework for responsive design and styling.
